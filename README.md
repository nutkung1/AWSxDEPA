## :bookmark_tabs: Run Locally

Clone the project

```bash
  git clone https://github.com/nutkung1/AWSxDEPA.git
```

Go to the project directory

```bash
  cd AWSxDEPA
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the python

```bash
  python run main.py
```
Start the server

```bash
  ngrok http 10000
```
